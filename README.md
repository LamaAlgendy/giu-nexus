# GIU Nexus — AI-Powered Career & Talent Platform

A MERN-stack web application that helps university students find internships and jobs,
and gives recruiters an AI-assisted way to discover the right candidates. Built for the
Software Engineering course, Spring 2026, at the German International University.

## Team

- Lama Algendy

## Tech Stack

- MongoDB (via Mongoose)
- Express.js
- React
- Node.js
- Hugging Face Inference API (job classification, skill extraction, job matching)
